翻译 Head First Java

你能从这本书学到什么？

Head First Java 是一本完整描述Java和面向对象编程的书。这本书将教会你怎么用不一样的方法去学习Java语法和怎么去变成一个伟大的编程者。你将学会语言基础，泛型，线程，网络，分布式处理，你还会建造一个”sink the dot com”的游戏，还有网络之间的交流。

这本书有什么特殊的？

我们认为你的时间很有价值，所以使用关于认知科学方面的方式给你一个很好的学习体验，不是单纯的文字让你昏昏欲睡。

  1. 潜水

打破表面

我们会深入一点，我们会写一点代码，然后编译，然后运行。我们会谈论语法，循环，分支结构，然后看看什么让Java这么cool，你就不久就会开始编程。

Java会带你去一个新地方。从Java低调的发布1.02版本，Java用它友好的语法，面向对象的特点，存储管理，最好的一致性吸引着编程者们。一次编写随处运行强大功能的诱惑，爆发了一大批因为应对bug和体验慢如狗的速度以及那些受到平台限制的追随者。但是那是很久以前的事情了，如果你刚开始学习Java，你是幸运的。我们在前面的版本遇到了很多困难，但是你现在可以体验更快更加强大的Java。

Java工作的方式

我们目标是写一个应用（一个派对的邀请函）可以在你朋友的任何设备上工作

  1. 创建一个源文件（关于邀请函的源代码），使用已经建立连接的协议（在这个给例子中我们用的是Java）
  2. 用源码编译器运行文档，源码编译器检查错误，如果你的文件没有语法错误是正常的就会被编译。
  3. 编译器创造一个新的文件，将这个文档编码成Java的字节码文件。任何能够翻译或者解释这个字节码文件的设备都可以运行这个文件，字节码文件是与平台无关。
  4. 你的朋友不需要一个Java机器，而是要一个Java虚拟机（指软件）在他们的设备里面运行，虚拟机可以读取并运行字节码文件。

你在Java里面做什么

你将输入一个源文件，使用编译器编译它，然后在Java虚拟机里面运行字节码文件

  1. 输入你的源码文件：例如 Party.java
  2. 通过javac（一个编译应用）编译Party.java，如果没有错误，你会得到另一个文件：Party.class，编译器生成的文件由字节码组成。
  3. 编译后的代码：Party.java
  4. Partyy.java字节码文件通过Java虚拟机（JVM）运行，JVM将字节码文件翻译成底层平台能够理解的文件。然后运行。

git的坑

Git 常用命令

git init 初始化本地仓库，把本地文件做成git可以管理的仓库 
git status 
git add .  相当于放缓存里面
git commit -m  "Message  "
git remote add origin #####  添加远端地址
git push -u origin master  推过去
git commit  --amend --no--edit  追加修改，还是先git add . 
git push

Giit上搜索技巧

精确名字 ：in:name ####
火热程度 ：stars:>## 或者=,< forks:也行
详情 ：in:readme in:description language:java
更新时间 ：pushed: 比如 >2019-03-0
